AddBWKSNotifyEventHandler

ブレイルワークスイベントハンドラの登録

HAGSHOOK AddBWKSNotifyEventHandler(
  AGSEventHandler aEventHandler
)

ブレイルワークス ピンディスプレイからの通知イベントが発生した時に呼ばれる、コールバック関数を登録します。

※コールバック関数は登録した順に呼び出されます。呼び出し順を変更するにはChangeEventHandlerLevel関数をコールします

引 数

aEventHandler

イベントコールバック関数

戻り値

HAGSHOOK

フックハンドル。イベントハンドラの登録解除(RemoveBWKSNotifyEventHandler)に必要

参照

DefEvent
RemoveBWKSNotifyEventHandler


イベントコールバック関数 


AGSEVENT_RET (CALLBACK *AGSEventHandler)( 
  AGSEVENTTYPE aEventType,
  INT_PTR aParam
)

引 数

aEventType

AGSEVENTTYPE_BWKSNOTIFYEVENTが設定されます。

aParam

LPBWKSNOTIFY_INFO


typedef struct{
    UINT		Id;    //	通知コード
    BWKSNMHDR*	nmdata;	//	ブレイルワークスデータ
    BYTE*		nmd;
}BWKSNOTIFY_INFO,*LPBWKSNOTIFY_INFO;

詳細はBOWKS.h参照

Id 通知コード。BWKS_REGMES_NOTIFYメッセージのWPARAM値
nmdata 通知コード別の構造体ポインタ
nmd  

戻り値

AGSEVENTRET_STOP 以降の処理を中断する
AGSEVENTRET_AGSNEXT 以降の処理を続行する
AGSEVENTRET_APIPROC AGS処理を中断する。次のコールバック関数がある場合はコールし、AGS処理は次のコールバック関数に任せる
AGSEVENTRET_STOPNEXTHOOK 次のコールバック関数をパスする。AGS処理は行う
AGSEVENTRET_PREVRESULT 前のコールバック関数の戻り値を採用する
