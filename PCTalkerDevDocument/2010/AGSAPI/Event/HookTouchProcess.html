HookTouchProcess

タッチイベントハンドラの登録


BOOL HookTouchProcess( 
      AGSEventHandler aEventHandler
)

タッチイベントが発生した時に呼ばれる、コールバック関数を登録します。

引数

aEventHandler

コールバック関数

戻り値

TRUE=成功　FALSE=失敗

タッチイベントコールバック関数 


  AGSEVENT_RET (CALLBACK *AGSEventHandler)(
        AGSEVENTTYPE aEventType,
        INT_PTR aParam
    )

引 数

aEventType

AGSEVENTTYPE_TOUCHACTNOTIFYが設定されます。

aParam

タッチ情報が渡されます


typedef struct{
    HWND hWnd;	  	      	//	ウインドウ
    LPTOUCHACT_NOTIFYINFO pTouchInf;	//	タッチ情報(KsTouch.h)
    BOOL	IsHelpGuide;		//	ヘルプガイド中か
    wchar_t* pHelpGuideString;		//	ヘルプガイド文字列(※アドインがセットする)
 }*LPTOUCHNOTIFYINFO;

戻り値

AGSEVENTRET_STOP PCTalker標準ののタッチ処理をしない
AGSEVENTRET_PREVRESULT PCTalker標準のタッチ処理を行う

参照

UnhookTouchProcess
