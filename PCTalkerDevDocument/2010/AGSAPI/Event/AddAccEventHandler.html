AddAccEventHandler

アクセシビリティイベントハンドラの登録


HAGSHOOK AddAccEventHandler( 
      AGSAccEventHandler aEventHandler,
      DWORD aEventId
)

AGSのアクセシビリティベントが発生した時に呼ばれる、コールバック関数を登録します。

※コールバック関数は登録した順に呼び出されます。呼び出し順を変更するにはChangeEventHandlerLevel関数をコールします

引 数

aEventHandler

イベントコールバック関数

戻り値

HAGSHOOK

フックハンドル。イベントハンドラの登録解除(RemoveAccEventHandler)に必要。

参照

DefEvent
RemoveAccEventHandler


イベントコールバック関数 


AGSEVENT_RET (CALLBACK *AGSAccEventHandler)( 
  AGSEVENTTYPE aEventType,INT_PTR aParam 
)

引 数

aEventType

アクセシビリティイベント時、AGSEVENTTYPE_ACCEVENTが設定されます。

aParam

イベントの付加情報


  typedef struct{
      DWORD	aEventId;	//	アクセシビリティイベントID(MSAAID)
      HWND	hWnd;
      DWORD	ObjectId;
      DWORD	ChildId;
  }*LPAGSACCEVENT_INFO;

戻り値

AGSEVENTRET_STOP 以降の処理を中断する
AGSEVENTRET_AGSNEXT 以降の処理を続行する
AGSEVENTRET_APIPROC AGS処理を中断する。次のコールバック関数がある場合はコールし、AGS処理は次のコールバック関数に任せる
AGSEVENTRET_STOPNEXTHOOK 次のコールバック関数をパスする。AGS処理は行う
AGSEVENTRET_PREVRESULT 前のコールバック関数の戻り値を採用する
