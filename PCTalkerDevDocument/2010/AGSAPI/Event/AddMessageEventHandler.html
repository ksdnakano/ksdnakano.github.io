AddMessageEventHandler

メッセージイベントハンドラの登録


HAGSHOOK AddMesageEventHandler(
    AGSMessageEventHandler aEventHandler,
    UINT aMessageNo
)

特定のメッセージ番号のメッセージイベントが発生した時に呼ばれる、コールバック関数を登録します。

※コールバック関数は登録した順に呼び出されます。呼び出し順を変更するにはChangeEventHandlerLevel関数をコールします

引 数

aEventHandler

コールバック関数

aMessageNo

フックするメッセージ番号

戻り値

HAGSHOOK

フックハンドル。フックハンドル。イベントハンドラの登録解除(RemoveMessageEventHandler)に必要。

参照

DefEvent
RemoveMessageEventHandler


イベントコールバック関数 


  AGSEVENT_RET (CALLBACK *AGSMessageEventHandler)( 
    AGSEVENTTYPE aEventType,
    INT_PTR aParam 
  )

引 数

aEventType

AGSEVENTTYPE_MESSAGEが設定されます。

aParam

メッセージ情報が渡されます


  typedef struct{
      MSG             msg;         //	メッセージ構造体
      WPARAM          wparamHook;  //	フック関数のWPARAM
      AGSMESSAGE_TYPE typeMessage; //	メッセージタイプ
  }*LPAGSMESSAGE_INFO;

戻り値

AGSEVENTRET_STOP 以降の処理を中断する
AGSEVENTRET_AGSNEXT 以降の処理を続行する
AGSEVENTRET_APIPROC AGS処理を中断する。次のコールバック関数がある場合はコールし、AGS処理は次のコールバック関数に任せる
AGSEVENTRET_STOPNEXTHOOK 次のコールバック関数をパスする。AGS処理は行う
AGSEVENTRET_PREVRESULT 前のコールバック関数の戻り値を採用する
