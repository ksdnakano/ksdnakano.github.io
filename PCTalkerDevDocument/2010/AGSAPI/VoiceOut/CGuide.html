１文字読み　CGuide

void CGuideA( const char* aReadText, int aMode, DWORD aFlags );
void CGuideW( const wchar_t* aReadText, int aMode,DWORD aFlags );

文字列をカーソル文字読みの読み方で読み上げます。

引 数

aReadText 読み上げ文字列
aMode

読み方モード

HIWORD(aMode) 0=カーソル文字の読み方で読み上げ
1=日本語変換の読み方で読み上げ
LOWORD(aMode) 文字読み上げモード
aFlags 読み上げ制御フラグの組み合わせ
