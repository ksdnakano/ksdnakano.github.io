フォーカスインしたウインドウ設定　PinFocus

BOOL PinFocusA( HWND aWnd, const char_t* aLabelString, uinsinged int aCtrlStyle, FARPROC aNotifyCallback, LONG_PTR aUserParam );
BOOL PinFocusW( HWND aWnd, wchar_t* aLabelString, uinsinged int aCtrlStyle, FARPROC aNotifyCallback, LONG_PTR aUserParam );
カスタムコントロール等をピンディスプレイ対応するために、フォーカスされたときに、ピンディスプレイに登録します。

引 数

aWnd ウインドウハンドル
aLabelString

コントロールのラベル文字列(タイトル)

下の エディットコントロールの例では、ラベル文字列="ハンドル(A):"

 
aCtrlStyle コントロールの種類とスタイルを指定します。
aNotifyCallback ピンディスプレイからの通知を受け取るコールバック関数を指定します
LONG_PTR コールバック関数に渡される任意の値を指定します。

※この関数呼び出し以降、PReadやVoiceGuideでは、ピンディスプレイ表示されなくなります。
　ピンディスプレイ出力を行うには、PinEDWrite、またはPinWriteをコールします。
