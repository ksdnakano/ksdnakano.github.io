文字列をピンディスプレイ出力　PinWrite

BOOL PinWriteA( const char* aString, unsinged int aInfoFlags, int aSubInfo );
BOOL PinWriteW( const wchar_t* aString, unsinged int aInfoFlags, int aSubInfo );
エディット型以外のコントロールの文字列をピンディスプレイに表示します。
※この関数をコールする前に、PinFocusでコントロールを登録しておく必要があります。

引 数

aString 表示する文字列
aInfoFlags

表示詳細区分
aSubInfo 追加情報

戻り値

BOOL
TRUE=成功
FALSE=失敗
