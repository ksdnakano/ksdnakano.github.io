ピンディスプレイステータスセルの表示　PinStatusCell

BOOL PinStatusCellA( const char* aStatusCell, const char* aHelp );
BOOL PinStatusCellW( const wchar_t* aStatusCell, const wchar_t* aHelp );
ピンディスプレイのステータスセル文字列をイに表示します。
※この関数をコールする前に、PinFocusでコントロールを登録しておく必要があります。

引 数

aStatusCell 表示するステータス文字列
aHelp

ステータスセルのヘルプ説明文

戻り値

BOOL
TRUE=成功
FALSE=失敗
