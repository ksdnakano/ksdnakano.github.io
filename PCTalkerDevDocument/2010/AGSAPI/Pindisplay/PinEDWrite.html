編集行をピンディスプレイ出力　PinEDWrite

BOOL PinEDWriteA( const char* aLineString, unsinged int aFlags, int aStartChar, int aEndChar, int aLineNumber, int aPageNumber, unsigned int aAttributes );
BOOL PinEDWriteW( const wchar_t* aLineString, unsinged int aFlags, int aStartChar, int aEndChar, int aLineNumber, int aPageNumber, unsigned int aAttributes );
エディット型コントロールの編集行をピンディスプレイに表示します。
※この関数をコールする前に、PinFocusでコントロールを登録しておく必要があります。

引 数

aLineString 表示するカーソル行文字列
aFlags

文字列のデータ区分
aStartChar 開始カーソル位置インデックス(0～)
aEndChar 終了カーソル位置インデックス(0～)
aLineNumber カーソル位置の行番号(1～) ※必須
aPageNumber カーソル位置のページ番号(1～) ※エディタのようにページが存在しない場合は0
aAttributes カーソル位置のアトリビュート

戻り値

BOOL
TRUE=成功
FALSE=失敗
