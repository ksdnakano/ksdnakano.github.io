CKSTouch::SetProcTouchSettings

プロセスのタッチ感度の設定


void CKSTouch::SetProcTouchSettings(
    LPTOUCH_SETTINGS pTouchSettings
)

タッチジェスチャを検知する感度等の設定値をセットします。

※この関数で設定する値は、プロセスのみの設定値です。
　他のプロセスには影響を与えません。

引数

pTouchSettings

設定値


typedef struct{
	UINT uTapIntervalMM;		//　タップ間隔(mmsec)
	UINT uLongTapTimeMM;		//　ロングタップ時間(mmsec)
	LONG nDriftMargin;		//　すべりマージン(mm*10)
	LONG nFlickDistanceMin;		//　フリック最低移動距離(mm*10)
	LONG nFlickCancelDistance;	//　フリックキャンセル距離感度(mm*10)
	LONG nPressSliderDistance;	//　プレススライダーコントロール切替距離(mm*10)
}TOUCH_SETTINGS,*LPTOUCH_SETTINGS;
