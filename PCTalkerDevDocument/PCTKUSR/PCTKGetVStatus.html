PCTKGetVStatus

音声発声状態の取得


BOOL PCTKGetVStatus( void )

戻り値

音声発声中であればTRUEを返し、発声中でなければ、FALSEを返します。

解説

音声の発声状態を返します。


注意:

PCTKGetVStatusは、SAPI音声エンジンにエラーが発生した場合もTRUEを返します。
次のコードのような音声待ちループを書くと永久ループに陥る可能性があります。


  while(PCTKGetVStatus()){
    ;
    ;
  }

安全のために、キーやマウス入力が発生した場合にループから抜ける処理、タイムアウト処理を書くことを推奨します。
