PCTKPinFocus

フォーカスインしたウインドウ設定


BOOL PCTKPinFocusA( 
  HWND aWnd,
  const char_t* aLabelString,
  uinsinged int aCtrlStyle,
  FARPROC aNotifyCallback,
  LONG_PTR aUserParam
)
 BOOL PCTKPinFocusW(
  HWND aWnd,
  wchar_t* aLabelString,
  uinsinged int aCtrlStyle,
  FARPROC aNotifyCallback,
  LONG_PTR aUserParam
)

引数

aWnd

ウインドウハンドル

aLabelString

コントロールのラベル文字列(タイトル)

aCtrlStyle

コントロールの種類とスタイルの組み合わせを指定します。

aNotifyCallback

ピンディスプレイからの通知を受け取るコールバック関数

aUserParam

任意の値。コールバック関数に渡されます。

戻り値

TRUE=成功

FALSE=失敗

解説

カスタムコントロール等をピンディスプレイ対応するために、フォーカスされたときに、ピンディスプレイに登録します。

aNotifyCallbackにコールバック関数を渡すと、ピンディスプレイからのコマンド要求通知を受け取ることができます。

※この関数呼び出し以降、PCTKPReadやPCTKPReadEx、PCTKVoiceGuideでは、ピンディスプレイ表示されなくなります。
ピンディスプレイ出力を行うには、PCTKPinEDWrite、またはPCTKPinWriteをコールします。
