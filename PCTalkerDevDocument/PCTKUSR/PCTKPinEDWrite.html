PCTKPinEDWrite

ピンディスプレイへ編集行を表示


BOOL PCTKPinEDWriteA( 
  const char* aString,
  unsinged int aFlags,
  int aStartChar,
  int aEndChar,
  int aLineNumber,
  int aPageNumber,
  unsigned int aAttributes
 )
BOOL PCTKPinEDWriteW( 
  const wchar_t* aString,
  unsinged int aFlags,
  int aStartChar,
  int aEndChar,
  int aLineNumber,
  int aPageNumber,
  unsigned int aAttributes
)

引数

aString

表示する編集１行の文字列

aFlags

文字列のデータ区分

PPED_NABCC_DATA aStringはNABCC行データ
PPED_CARET aStringは通常の行文字列でaStartCharに指定した値はカーソル位置
PPED_RANGE aStringは通常の行文字列でaStartCharからaEndCharを範囲選択中
aStartChar

開始カーソル位置インデックス(0～)

aEndChar

終了カーソル位置インデックス(0～)

aLineNumber

カーソル位置の行番号(1～) ※必須

aPageNumber

カーソル位置のページ番号(1～) ※エディタのようにページが存在しない場合は0

aAttributes

カーソル位置のアトリビュート


戻り値

TRUE=成功

FALSE=失敗

解説

ピンディスプレイへ編集行文字列を表示します。

※この関数呼び出しによる音声は出力しません。
