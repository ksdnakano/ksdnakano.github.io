PCTKBeep

ビープ音を鳴らす


void PCTKBeep( 
  int BeepType=PCTK_BEEP1KHz,
  UINT mmsec=15,
  int MachinType=PCTK_MACHIN_AUTO
);

引数

BeepType

以下のビープ音の種類を指定します。

PCTK_BEEP_500KHz
1		 500KHzのビープ音
PCTK_BEEP_1KHz
2		 1KHzのビープ音
PCTK_BEEP_2KHz
3		 2KHzのビープ音
mmsec

ビープ音の長さ(ミリ秒)を指定します。

MachinType

PCの機種を設定します。

(注)PCTalkerのバージョンがVer.3.3以降であれば、この値は無視されます。PCTalker Ver.3.3以降、ハードウエアビープからwavに変更されたため。

PCTK_MACHIN_AUTO
0		 自動判定
PCTK_MACHIN_NEC98
1		 NEC98シリーズ
PCTK_MACHIN_DOSV
2		 DOS/Vシリーズ

解説

ビープ音を鳴らします。(PCTalkerのバージョンが3.3より前であれば、このビープ音は、ハードウェアに依存します。)
