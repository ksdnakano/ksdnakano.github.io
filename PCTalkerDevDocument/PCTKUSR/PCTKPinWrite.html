PCTKPinWrite

ピンディスプレイへ表示

BOOL PCTKPinWriteA( 
  const char* aString,
  unsinged int aInfoFlags,
  int aSubInfo
)
BOOL PCTKPinWriteW( 
  const wchar_t* aString,
  unsinged int aInfoFlags,
  int aSubInfo
)

引数

aString

表示する文字列

aInfoFlags

表示詳細区分

aSubinfo

追加情報。
aInfoFlagsがPPW_LI_TREEVIEWの場合、ツリービュー項目の階層レベルを指定します。


戻り値

TRUE=成功

FALSE=失敗

解説

ピンディスプレイへ文字列を表示します。

※この関数呼び出しによる音声は出力しません。
